# 状态和生命周期

考虑前面章节中的时钟的例子。

到目前位置，我们仅学习了一种更新 UI 的方式。

我们调用`ReactDOM.render()`来改变渲染输出：

```
function tick() {
  const element = (
    <div>
      <h1>Hello, world!</h1>
      <h2>It is {new Date().toLocaleTimeString()}.</h2>
    </div>
  );
  ReactDOM.render(
    element,
    document.getElementById('root')
  );
}

setInterval(tick, 1000);
```
